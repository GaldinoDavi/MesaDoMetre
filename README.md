# Mesa do Mestre

Companion web para mestres de RPG de mesa gerenciarem campanhas, NPCs e sessões, com
auxílio de criação de conteúdo por IA. Visão completa em [`PRD.md`](./PRD.md); spec da
onda em andamento em [`docs/specs/onda-0.md`](./docs/specs/onda-0.md).

## Getting Started

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Stack

Next.js (App Router) + shadcn/ui · Supabase (Postgres + Auth + Storage) · Vercel · API da Anthropic
